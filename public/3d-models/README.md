Place exported Spline scene files here.

How to export from Spline:
1. Open your scene in https://app.spline.design
2. Export → Download bundle
3. Unzip and copy `scene.splinecode` and any assets into this folder

Example:
- `/public/3d-models/scene.splinecode`

When using `@splinetool/react-spline`, reference `/3d-models/scene.splinecode` as the `scene` prop.
